# Проект Новостной сайт.
Разработан в качестве практического задания на курсе Fullstack веб-разработчик на JavaScript и PHP. Модуль - Базовый CSS
## Используемые технологии
* HTML
* CSS
## Цель проекта
- создание новостной страницы с использованием стилей CSS.
## Результат
* При помощи CSS кода, было успешно сделано позиционирование блоков на странице. Получилось стилизовать текст. Также в работе были применены селектор псевдокласса, для работы с блоками информации, и селектор псевдоэлемента, для работы с текстом.
## Дальнейшие планы
- содание динамики на странице при помощи JavaScript
